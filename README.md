# Data Science Portfolio Dashboard

An interactive Streamlit dashboard showcasing three end-to-end data science projects.

## Projects

### 🌍 World Happiness Explorer
Exploratory analysis of global happiness across 150+ countries from 2015–2019.
Interactive filters by year and region with correlation heatmap.

### 🏨 Hotel Cancellation Predictor
Random Forest classifier predicting hotel booking cancellations with 87% accuracy.
Live predictions based on user-input booking details.

### 🚢 Hampton Roads AIS Traffic
Time series analysis of 2.5 million AIS vessel pings near Naval Station Norfolk (2023).
Traffic patterns by hour, day, and month.

## Running Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
